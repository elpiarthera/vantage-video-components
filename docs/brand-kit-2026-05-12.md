## 2. Color palette (live confirmed)

Extracted from production CSS `:root`:

| Token | Hex | Use |
|---|---|---|

| `--prose-body` | `#e5e7eb` | body text on dark |
| `--border` | `#1f2937` | dividers |
| `--heading` | `#ffffff` | white headings on dark (sparingly — see typography §3) |

**Rule**: amber `#f59e0b` is the only accent. Never flood. One light source per frame, one accent line per overlay. Other warm tones (`#c9b99a` secondary highlight, `#e8e0d0` body warm-white) appear in the typography spec — use those instead of pure white for body copy on near-black to avoid clinical contrast.

## 3. Typography

**Live stack (Georgia + system-ui)** is canonical for the site itself. **Cormorant Display upgrade** is approved for reels only, with hard constraints.

| Level | Size | Font | Weight | Letter-spacing | Use |
|---|---|---|---|---|---|
| H1 reel title card | 80px | **Cormorant Display** | Bold 700 | −2% | reel hero title |
| H2 reel subhead | 56px | **Cormorant Display** | SemiBold 600 | −1.5% | reel kicker |
| H3 callout | 40px | **Georgia** | Bold 700 | −0.5% | reel mid-frame call-out |
| Body | 24px | **Georgia** | Regular 400 | 0% | reel body lines |
| Caption / metadata | 18px | **system-ui** | Regular 400 | −0.5% | URL, attribution, end-card |

**Hard constraints** (AD verdict §10):
- Cormorant Display ONLY at sizes ≥56px. Below 56px → Georgia mandatory (no exceptions). This preserves visual continuity when viewers arrive on the live site after the reel.
- Never use Cormorant below 32px (thin-stroke collapse).
- Never use pure `#ffffff` on `#0a0a0a` for body text — use `#e8e0d0` (warm white).
- Color: amber `#f59e0b` for title accent line; `#ffffff` reserved for top-level heading; `#e5e7eb` or `#c9b99a` for secondary.

**Web import (reel-only, do not load on live site)**:
```
https://fonts.googleapis.com/css2?family=Cormorant+Garamond:ital,wght@0,300;0,400;0,600;0,700;1,400;1,600&family=Cormorant:ital,wght@0,300;0,400;0,600;0,700;1,400;1,600&display=swap
```

For Rho's reel renderer, prefer self-hosting `.woff2` from the Google Fonts GitHub repo to eliminate network dependency at render time.

Full spec retained in `reference/brand-kit-pre-audit/typography.md` (read with §10 constraints applied).
