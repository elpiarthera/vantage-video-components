## Description

`FadeBlur` dissolves two scenes through a peak of gaussian blur, creating the visual equivalent of a camera rack-focus between chapters. The blur rises on the outgoing scene while the incoming scene emerges from behind the same blur, so both share a single moment of maximum softness before sharpening back to full resolution.

Three operational modes give precise control: `cross` runs the full dissolve through the blur peak (the typical choice); `in` blurs only the incoming scene as it arrives; `out` blurs only the departing scene as it leaves. The optional amber bloom at the crossover point preserves the BRAND.md single-accent restraint while marking the seam.

## Installation

```bash
npx shadcn add https://raw.githubusercontent.com/elpiarthera/vantage-video-components/main/registry/fade-blur.json
```

The component lands in `remotion/transitions/fade-blur.tsx` and is yours to edit.

## Usage

```tsx
import { AbsoluteFill, Composition } from "remotion";
import { FadeBlur } from "@/remotion/transitions/fade-blur";

function SceneA() {
  return <AbsoluteFill style={{ background: "#0a0a0a" }} />;
}

function SceneB() {
  return <AbsoluteFill style={{ background: "#141414" }} />;
}

function MyComposition() {
  return (
    <FadeBlur
      direction="cross"
      durationFrames={20}
      blurRadius={12}
      accentColor="#f59e0b"
      easing="literary"
      transitionStart={35}
      from={<SceneA />}
      to={<SceneB />}
    />
  );
}

export function Root() {
  return (
    <Composition
      id="MyFadeBlur"
      component={MyComposition}
      durationInFrames={90}
      fps={30}
      width={1920}
      height={1080}
    />
  );
}
```

<Warning title="CSS filter in Remotion">
  `filter: blur()` animates the GPU compositing layer. Keep `blurRadius` at 20 or below during fast cuts (under 15 frames) — extreme blur on very short transitions can produce a visual strobe rather than a soft dissolve.
</Warning>
