## Description

`ZoomPulse` adds kinetic weight to a cut by pushing both the outgoing and incoming scenes through a shared scale peak. The outgoing scene scales up toward `scalePeak` as it approaches the cut, while the incoming scene starts at that same peak and decelerates back to `1.0`. The result is a lens-punch feel — as if the camera breathed in just before and out just after the edit.

Three direction modes fine-tune the effect: `punch` (both scenes traverse the scale peak together) is the signature look for high-energy product demos; `in` scales only the incoming scene; `out` scales only the departing scene. An amber radial flash at the exact cut frame marks the instant without persisting.

## Installation

```bash
npx shadcn add https://raw.githubusercontent.com/elpiarthera/vantage-video-components/main/registry/zoom-pulse.json
```

The component lands in `remotion/transitions/zoom-pulse.tsx` and is yours to edit.

## Usage

```tsx
import { AbsoluteFill, Composition } from "remotion";
import { ZoomPulse } from "@/remotion/transitions/zoom-pulse";

function SceneA() {
  return <AbsoluteFill style={{ background: "#0a0a0a" }} />;
}

function SceneB() {
  return <AbsoluteFill style={{ background: "#141414" }} />;
}

function MyComposition() {
  return (
    <ZoomPulse
      direction="punch"
      durationFrames={18}
      scalePeak={1.08}
      accentColor="#f59e0b"
      easing="snappy"
      transitionStart={36}
      from={<SceneA />}
      to={<SceneB />}
    />
  );
}

export function Root() {
  return (
    <Composition
      id="MyZoomPulse"
      component={MyComposition}
      durationInFrames={90}
      fps={30}
      width={1920}
      height={1080}
    />
  );
}
```
