## Description

`SlideReveal` translates two scenes along a shared axis — the departing frame slides out while the arriving frame slides in from the opposite edge. An `overlap` parameter controls how much the two scenes share the frame simultaneously: at `0` the outgoing scene fully exits before the incoming begins; at `1` they move in complete lock-step.

A thin amber streak races along the seam at peak transition velocity, marking the boundary between scenes without overwhelming the motion. The `snappy` default easing — Material standard `cubic-bezier(0.4, 0, 0.2, 1)` — suits fast product-reveal cuts where momentum carries narrative weight.

## Installation

```bash
npx shadcn add https://raw.githubusercontent.com/elpiarthera/vantage-video-components/main/registry/slide-reveal.json
```

The component lands in `remotion/transitions/slide-reveal.tsx` and is yours to edit.

## Usage

```tsx
import { AbsoluteFill, Composition } from "remotion";
import { SlideReveal } from "@/remotion/transitions/slide-reveal";

function SceneA() {
  return <AbsoluteFill style={{ background: "#0a0a0a" }} />;
}

function SceneB() {
  return <AbsoluteFill style={{ background: "#141414" }} />;
}

function MyComposition() {
  return (
    <SlideReveal
      direction="left"
      durationFrames={24}
      overlap={0.3}
      accentColor="#f59e0b"
      easing="snappy"
      transitionStart={36}
      from={<SceneA />}
      to={<SceneB />}
    />
  );
}

export function Root() {
  return (
    <Composition
      id="MySlideReveal"
      component={MyComposition}
      durationInFrames={90}
      fps={30}
      width={1920}
      height={1080}
    />
  );
}
```

<Warning title="Overflow clipping">
  `SlideReveal` sets `overflow: hidden` on the outer container. If your scenes use `position: fixed` or render outside the composition bounds, those elements will be clipped during the transition window.
</Warning>
