## Description

`CinematicCut` is an instantaneous scene change that earns its weight through a brief motion-blur compression on the outgoing frame and a single amber accent flash at the cut point. The visual cue is subconscious — viewers read it as a confident editorial decision rather than a technical transition. Use it when a dissolve would read as hesitant or a wipe would read as decorative.

The `literary` easing follows a slow-in / slow-out cubic-bezier that suits contemplative, narrative cuts. The `snappy` preset uses the Material standard curve for brisk chapter changes or UI-rhythm cuts inside a product reel.

## Installation

```bash
npx shadcn add https://raw.githubusercontent.com/elpiarthera/vantage-video-components/main/registry/cinematic-cut.json
```

The component lands in `remotion/transitions/cinematic-cut.tsx` and is yours to edit.

## Usage

```tsx
import { AbsoluteFill, Composition, Sequence } from "remotion";
import { CinematicCut } from "@/remotion/transitions/cinematic-cut";

function SceneA() {
  return <AbsoluteFill style={{ background: "#0a0a0a" }} />;
}

function SceneB() {
  return <AbsoluteFill style={{ background: "#141414" }} />;
}

function MyComposition() {
  return (
    <CinematicCut
      direction="horizontal"
      duration={20}
      accentColor="#f59e0b"
      easing="literary"
      transitionStart={45}
      from={<SceneA />}
      to={<SceneB />}
    />
  );
}

export function Root() {
  return (
    <Composition
      id="MyCinematicCut"
      component={MyComposition}
      durationInFrames={90}
      fps={30}
      width={1920}
      height={1080}
    />
  );
}
```

<Warning title="One accent per scene">
  Per BRAND.md restraint rules, the accent flash must be the sole amber element on any frame it occupies. If your `from` or `to` scenes already use `--vantage-accent`, lower the flash opacity or set `accentColor` to `"transparent"` to avoid competing warmth signals.
</Warning>
