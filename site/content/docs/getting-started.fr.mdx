## Installation

Les Vantage Video Components sont distribués via le registre shadcn. Chaque composant est un fichier autonome que vous possédez — pas de package opaque à mettre à jour.

### Prérequis

- Next.js 15+ avec App Router
- React 19+
- Remotion 4.0+
- CLI shadcn (`npx shadcn@latest`)

### Installer un composant

Utilisez le schéma d'URL du registre public :

```bash
npx shadcn@latest add https://raw.githubusercontent.com/elpiarthera/vantage-video-components/main/registry/<nom>.json
```

Remplacez `<nom>` par le slug du composant, par exemple :

```bash
npx shadcn@latest add https://raw.githubusercontent.com/elpiarthera/vantage-video-components/main/registry/cinematic-cut.json
```

### Tokens de marque

Tous les composants consomment la palette OKLCH définie dans `BRAND.md`. Ajoutez ces variables CSS à la feuille de style globale de votre projet :

```css
:root {
  --vantage-bg:               oklch(0.16 0 0);
  --vantage-surface:          oklch(0.18 0.008 290);
  --vantage-accent:           oklch(0.74 0.16 60);
  --vantage-muted:            oklch(0.65 0 0);
  --vantage-prose:            oklch(0.91 0 0);
  --vantage-warm-white:       oklch(0.93 0.01 80);
  --vantage-secondary-highlight: oklch(0.79 0.04 76);
  --vantage-border:           oklch(0.22 0 0);
  --vantage-heading:          oklch(1 0 0);

  --font-display: "Cormorant Display", Georgia, serif;
  --font-body:    Georgia, "Times New Roman", serif;
  --font-caption: system-ui, -apple-system, sans-serif;
}
```

### Règles typographiques

- `Cormorant Display` n'est autorisé qu'à **56px et au-dessus**. En dessous de 56px, utilisez `Georgia`.
- N'utilisez jamais `#ffffff` pur pour le corps de texte — utilisez `--vantage-warm-white` (`#e8e0d0`) à la place.
- Un seul point d'accentuation ambre par scène maximum.
