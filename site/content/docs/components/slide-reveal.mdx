## Description

`SlideReveal` translates its child from an off-screen position — up to `distance × 100%` along the chosen axis — into its final resting position, pairing the movement with a simultaneous opacity fade over the first half of the transition duration. This combined slide + fade entrance is the canonical Vantage technique for content elements: supporting text, data callouts, and caption layers.

Per the BRAND.md slide doctrine, `SlideReveal` is reserved for content elements and must never be applied to full-frame backdrops or Cormorant Display H1 headings. Combine with `FadeBlur` for a two-layer refined entrance. Direction supports all four axes: `left` (content enters from left), `right`, `up`, and `down`. The `distance` prop controls how far off-screen the starting position is, from `0` (no movement) to `1` (full viewport offset).

## Usage

```tsx
import { AbsoluteFill, Composition, Sequence } from 'remotion'
import { SlideReveal } from '@/remotion/transitions/slide-reveal'

function ContentScene() {
  return (
    <AbsoluteFill style={{ background: '#0a0a0a' }}>
      <Sequence from={10}>
        <SlideReveal
          direction="left"
          durationFrames={24}
          distance={1}
          easing="literary"
          startFrame={0}
        >
          <AbsoluteFill style={{ display: 'flex', alignItems: 'center', justifyContent: 'center' }}>
            <p style={{ color: '#e8e0d0', fontSize: 24 }}>Supporting text enters here</p>
          </AbsoluteFill>
        </SlideReveal>
      </Sequence>
    </AbsoluteFill>
  )
}

export function MyComposition() {
  return (
    <Composition
      id="SlideRevealDemo"
      component={ContentScene}
      durationInFrames={60}
      fps={30}
      width={1080}
      height={1920}
    />
  )
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `direction` | `"left" \| "right" \| "up" \| "down"` | `"left"` | The axis and direction from which the content enters the frame. |
| `durationFrames` | `number` (8–60) | `24` | Number of frames for the full slide+fade transition. |
| `distance` | `number` (0–1) | `1` | Fraction of the viewport dimension used as the off-screen starting offset. `1` = full width/height offset. |
| `easing` | `"literary" \| "snappy"` | `"literary"` | Easing curve for the translate animation. Opacity fade uses the same curve over the first 50% of duration. |
| `children` | `ReactNode` | Demo content | Content to slide into view. |
| `startFrame` | `number` | `0` | Frame at which the slide begins, relative to composition start. |
