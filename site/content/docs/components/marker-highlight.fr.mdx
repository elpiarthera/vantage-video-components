## Description

`MarkerHighlight` étend une bande de surlignage semi-transparent sur un mot cible comme s'il était dessiné par un marqueur feutre. La bande s'étend horizontalement de gauche à droite via `scaleX` (animation de transform uniquement, selon la doctrine de mouvement BRAND.md). Au fur et à mesure que le marqueur couvre le mot, la couleur du texte bascule vers `textShiftColor` — par défaut `--vantage-bg` (`#0a0a0a`) pour un renversement sombre sur ambré.

La couleur du marqueur est par défaut `oklch(0.74 0.16 60 / 0.35)` — l'accent ambré BRAND.md à 35% d'opacité, préservant la lisibilité tout en respectant la doctrine d'un seul accent par scène. L'attribut `aria-hidden="true"` est appliqué au span décoratif du marqueur afin que les lecteurs d'écran ne reçoivent que le texte brut.

Composé avec les primitives Remotion — `AbsoluteFill`, `useCurrentFrame`, `interpolate`, `interpolateColors` — sans l'API TransitionSeries de `@remotion/transitions`.

## Utilisation

```tsx
import { AbsoluteFill, Composition } from 'remotion'
import { MarkerHighlight } from '@/remotion/text-effects/marker-highlight'

function ScèneMarqueur() {
  return (
    <AbsoluteFill style={{ background: '#0a0a0a' }}>
      <MarkerHighlight
        text="L'avenir est écrit."
        highlightWord="avenir"
        markerColor="rgba(245,158,11,0.35)"
        textShiftColor="#0a0a0a"
        drawFrames={18}
        easing="literary"
        startFrame={0}
      />
    </AbsoluteFill>
  )
}

export function MaComposition() {
  return (
    <Composition
      id="MarkerHighlightDemo"
      component={ScèneMarqueur}
      durationInFrames={60}
      fps={30}
      width={540}
      height={960}
    />
  )
}
```

## Props

| Prop | Type | Défaut | Description |
|------|------|--------|-------------|
| `text` | `string` | — | Texte complet de la phrase. |
| `highlightWord` | `string` | — | Le mot à surligner. La première occurrence dans `text` est utilisée. |
| `markerColor` | `string` | `"oklch(0.74 0.16 60 / 0.35)"` | Couleur de remplissage du marqueur. Accent ambré à 35% d'opacité per BRAND.md Rule 1. |
| `textShiftColor` | `string` | `"#0a0a0a"` | Couleur vers laquelle bascule le texte du mot surligné au fur et à mesure du tracé. |
| `drawFrames` | `number` (8–40) | `18` | Nombre d'images pour que le marqueur se dessine entièrement sur le mot. |
| `easing` | `"literary" \| "snappy"` | `"literary"` | Courbe d'easing pour l'animation de tracé du marqueur. |
| `startFrame` | `number` | `0` | Image à laquelle le tracé du marqueur commence. |
