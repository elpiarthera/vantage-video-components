## Description

`FadeBlur` combine opacité et `blur()` CSS en une révélation éditoriale unique. En fondu entrant, l'opacité monte de 0 à 1 tandis que le flou se dissout de `blurMax` à 0 — le contenu se cristallise depuis un halo doux jusqu'à une netteté parfaite. En fondu sortant, le mouvement s'inverse. Cette entrée à double dimension est la technique Vantage canonique pour les couches de texte de corps et de sous-titres : elle évite la platitude d'un simple fondu d'opacité tout en restant plus légère qu'un glissement complet.

Les deux courbes d'easing sont issues du vocabulaire de mouvement BRAND.md : `literary` (`cubic-bezier(0.25, 0.46, 0.45, 0.94)`) pour les entrées de contenu narratif, `snappy` (`cubic-bezier(0.4, 0, 0.2, 1)`) pour les transitions de chrome UI. Composée avec les primitives Remotion — `AbsoluteFill`, `useCurrentFrame`, `interpolate` — et non l'API TransitionSeries de `@remotion/transitions`.

## Utilisation

```tsx
import { AbsoluteFill, Composition, Sequence } from 'remotion'
import { FadeBlur } from '@/remotion/transitions/fade-blur'

function ScèneContenu() {
  return (
    <AbsoluteFill style={{ background: '#0a0a0a' }}>
      <Sequence from={0}>
        <FadeBlur
          direction="in"
          durationFrames={20}
          blurMax={12}
          easing="literary"
          startFrame={0}
        >
          <AbsoluteFill style={{ display: 'flex', alignItems: 'center', justifyContent: 'center' }}>
            <h1 style={{ color: '#e5e7eb', fontSize: 56 }}>Votre Contenu</h1>
          </AbsoluteFill>
        </FadeBlur>
      </Sequence>
    </AbsoluteFill>
  )
}

export function MaComposition() {
  return (
    <Composition
      id="FadeBlurDemo"
      component={ScèneContenu}
      durationInFrames={60}
      fps={30}
      width={1080}
      height={1920}
    />
  )
}
```

## Propriétés

| Prop | Type | Défaut | Description |
|------|------|--------|-------------|
| `direction` | `"in" \| "out"` | `"in"` | Direction du fondu : `"in"` révèle le contenu, `"out"` le dissimule. |
| `durationFrames` | `number` (8–60) | `20` | Nombre de frames sur lesquelles le fondu+flou se joue. |
| `blurMax` | `number` (0–30) | `12` | Rayon de flou maximal en pixels au début d'un fondu entrant (ou à la fin d'un fondu sortant). |
| `easing` | `"literary" \| "snappy"` | `"literary"` | Courbe d'easing appliquée à l'interpolation de l'opacité et du flou. |
| `children` | `ReactNode` | Contenu démo | Contenu à envelopper dans la transition fondu-flou. |
| `startFrame` | `number` | `0` | Frame à laquelle la transition démarre, relative au début de la composition. |
